Small Potato - xboard/winboard compatible chess engine

A chess engine, like Crafty or gnuchess or Fruit, but much weaker. It would probable rate around 2100 on five-minute games. It currently does not run on 64-bit OSes

Window binaries: http://organicrobot.com/smallpotato/smallpotato061.zip
Book:

  • 32-bit for versions up to 0.6.1: http://organicrobot.com/smallpotato/book32-0.6.1.zip or http://organicrobot.com/smallpotato/book32-0.6.1.opn.bz2
  • 64-bit for versions 0.6.900 and up: http://organicrobot.com/smallpotato/book64.opn.bz2 Uncompress the book and put in the same directory as the binary.

To run using xboard as a frontend, use
xboard -fcp smallpotato
To run it through WinBoard, you will need to add something like the following to your winboard.ini:
/fcp smallpotato /fd "full\path\to\smallpotato\directory"
/scp smallpotato /sd "full\path\to\smallpotato\directory"

Change engine settings by editing sp.rc

For the chess-engine enthusiast, it implements:

  • Negascout
  • Transposition table
  • Quiescent search
  • Null-move pruning
  • History move ordering
  • Run-time configurable value tables
