Small Potato (aka unches)

An attempt at a chess program. Sitting at around 1900 on 3-minute chess on FICS (nick is unches).

Features
  • Good compatibility with the Chess Engine Communication Protocol.
  • Runtime definable value tables
  • Negascout/Alpha-beta/transposition table/quiescent search/null-move pruning/history move ordering
  • Lots of bugs for your debugging exercises

Download it here (sourceforge). GPL license. Latest version is 0.6.1

The opening book is packaged separately since it is relatively big (3 MB compressed). This should be put in the same directory as smallpotato. There has been no changes in book format yet, so any book from old releases should work just as well. Windows binaries are also available for most releases

More info at the project page on sourceforge
